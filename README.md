# Snake

Classic Snake game implemented with vanilla HTML/CSS/JS.

## Run locally

1. Start a static file server from the repo root:
   - `python3 -m http.server 4173`
2. Open: `http://localhost:4173`

## Run tests

- `npm test`

## Manual verification checklist

- [ ] Snake moves automatically on a grid.
- [ ] Arrow keys and WASD change direction (without instant reverse).
- [ ] On-screen directional buttons move the snake.
- [ ] Eating food grows snake and increases score.
- [ ] Hitting wall or self ends game and shows "Game Over".
- [ ] Space toggles pause/resume.
- [ ] `R` key and Restart button reset game state.
